Dropping a Trigger
==================

.. important::

   Dropped triggers cannot be recovered. Exercise caution when performing this operation.

Procedure
---------

#. On the top menu bar, choose **Database Management**. On the displayed **Objects** page, select **Triggers**, locate the target trigger, and click **Drop** in the **Operation** column.
#. In the displayed dialog box, click **Yes**.
